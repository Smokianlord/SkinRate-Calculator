## Features

- Compare different transfer rates and cashout options
- Auto-calculate fees and profit margins
- Uses Bangladeshi taka symbol: `৳`
- No comma formatting in taka amounts
- Colorful 3D-style button UI
- Simple desktop `.exe` app

## Download

Go to the **Releases** section and download:

```text
SkinRate Calculator.exe
```

Then double-click it to open the app.

> Note: The `.exe` version may take a few seconds to open the first time.
